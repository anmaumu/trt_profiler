# trt_profiler ドキュメント

このディレクトリには、現在の実装を理解して使うための設計文書とチュートリアルを置きます。

## 文書一覧

- [application_spec.md](application_spec.md): アプリケーション仕様、CLI、config、入出力、レポート仕様
- [uml.md](uml.md): Mermaid UML による class 図、sequence 図、component 図
- [tutorial_user_model.md](tutorial_user_model.md): ユーザー自身の ONNX モデルで評価する手順
- [design.md](design.md): 初期設計から詳細方針まで含む長めの設計仕様書

## 読む順番

初めて使う場合:

2. [tutorial_user_model.md](tutorial_user_model.md)
3. [uml.md](uml.md)
